# Quarto documents

`using_quarto()` determines whether code is being run within a Quarto
document

## Usage

``` r
using_quarto()
```

## Value

A logical value

## Note

The `is_using_quarto()` function in the
[quarto](https://cran.r-project.org/package=quarto) package can be used
to determine whether your are in a quarto project.

## Examples

``` r
using_quarto()
#> [1] FALSE
```
